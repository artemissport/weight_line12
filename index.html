問題從來不是「要不要吃澱粉」

首先，糖跟醣到底差在哪

很多人把這兩個搞混。看到營養標籤上寫「碳水化合物（醣）」就嚇到：「天啊！這個糖好多！我不能吃！」然後什麼澱粉都不敢碰。

但其實，糖跟醣是不一樣的東西。

糖（糖）

就是甜的那個糖

  • 精緻糖、砂糖、果糖、蔗糖
  • 手搖飲裡的糖
  • 蛋糕、餅乾、糖果裡的糖
  • 果汁裡的果糖

這些要減少，沒錯。

醣（醣）

碳水化合物的總稱

  • 糖（上面說的那些）
  • 澱粉（飯、麵、地瓜）
  • 膳食纖維（蔬菜、全穀類裡的）

簡單記：

糖是醣的一種，但醣不等於糖

就像：水果是食物的一種，但食物不等於水果。糖只是醣這個大家族裡的一員。

為什麼要搞懂這個？

因為營養標籤上寫的「碳水化合物（醣）」，裡面包含了：

  • ✓ 糖（要減少的）
  • ✓ 精緻澱粉（要注意的）
  • ✓ 全穀澱粉（要吃的）
  • ✓ 膳食纖維（要多吃的）

所以不能看到「醣」就以為都是壞東西。重點是要分辨「好的醣」跟「壞的醣」。

特別說明：天然糖 vs 添加糖

天然糖（水果裡的果糖）：有纖維包著、有維生素、礦物質，適量吃OK

添加糖（手搖飲、蛋糕裡的糖）：沒有纖維、沒有營養，要減少

  • ✓ 吃一顆蘋果 → OK（天然糖+纖維）
  • ✗ 喝一杯蘋果汁 → 不好（天然糖但纖維被去掉了）
  • ✗ 喝珍奶 → 更不好（添加糖+沒纖維）

不要矯枉過正，連水果都不敢吃。新鮮水果是好的，適量吃沒問題。

真正的問題：好澱粉 vs 壞澱粉

澱粉本身沒有錯。錯的是「精緻加工」和「失去纖維」。

國外有本書叫「好碳水、壞碳水」，講的就是這件事。讓我來教你怎麼分辨：

壞澱粉（精緻澱粉）

  • 白吐司、白麵包、甜麵包
  • 白麵條
  • 蛋糕、餅乾、甜甜圈
  • 珍珠、粉圓、芋圓
  • 勾芡的醬汁
  • 炒飯、炒米粉
  • 麵包粉裹的炸物
為什麼不好：
  • 纖維完全被去掉了
  • 血糖快速飆升
  • 容易餓、容易想再吃
  • 多餘的糖變成脂肪囤積

好澱粉（原型澱粉）

  • 糙米、五穀飯、胚芽米
  • 地瓜、芋頭、南瓜、山藥
  • 全麥麵包（真的全麥）
  • 燕麥、藜麥、小米
  • 豆類（紅豆、綠豆、鷹嘴豆）
  • 玉米（水煮的）
  • 馬鈴薯（蒸的、烤的）
為什麼好：
  • 保留完整纖維
  • 血糖上升緩慢平穩
  • 飽足感強、持久
  • 幫助腸道健康

特別說明：白飯的定位

你可能會問：「那白飯呢？」

白飯其實不算壞澱粉。雖然糙米確實比白飯更好（纖維更多），但白飯比起蛋糕、麵包、珍奶這些，已經好很多了。

白飯的關鍵不是好壞，而是怎麼吃：

  • ✗ 單吃一碗白飯 → 血糖暴衝
  • ✗ 一大碗飯配一點菜 → 還是不夠好
  • ✓ 白飯 + 大量蔬菜 + 蛋白質 → 這樣吃就OK
  • ✓ 半碗白飯 + 大半碗菜 + 一個手掌蛋白質 → 更好

為什麼？因為「混合進食效應」

這是營養學的專業概念。當你把白飯混著蔬菜、蛋白質、油脂一起吃時，整餐的升糖指數（GI值）會被拉低。

單吃白飯 GI = 80（很高）

但白飯+菜+肉一起吃，整餐的 GI 可能只有 50-60（中等）

重點是「複合著吃」，不是「單吃一樣」

關鍵差別：膳食纖維

這是好澱粉跟壞澱粉最大的不同。膳食纖維就像血糖的守門員。

好澱粉（有守門員）

纖維會包住澱粉
讓糖分緩慢釋放到血液
血糖平穩上升
胰島素適量分泌
能量穩定供應
不會一下餓一下飽

壞澱粉（沒有守門員）

沒有纖維保護
糖分快速衝進血液
血糖暴衝
胰島素拼命分泌
血糖又快速下降
你又餓了，想再吃
惡性循環

為什麼好澱粉不會讓你胖

很多人以為：澱粉 = 熱量 = 會胖

但其實重點不只是熱量。更重要的是「你的身體如何處理這些熱量」。

當你吃精緻澱粉

  1. 血糖快速飆高（像坐雲霄飛車）
  2. 胰島素大量分泌（緊急處理）
  3. 把糖快速塞進細胞（變脂肪儲存）
  4. 血糖又快速下降
  5. 你又餓了，而且特別想吃甜的
  6. 重複循環

長期下來：胰島素累壞了、身體變得容易囤脂肪、代謝越來越差、越來越難瘦

當你吃好澱粉

  1. 血糖緩慢上升（平穩的斜坡）
  2. 胰島素適量分泌（從容處理）
  3. 能量平穩供應給細胞使用
  4. 血糖維持穩定
  5. 飽足感持續3-4小時
  6. 不會一直想吃

長期下來：胰島素敏感度維持良好、身體願意釋放脂肪、代謝正常運作、自然就瘦了

好澱粉還有一個秘密武器：自動煞車機制

你可能會問：「那好澱粉吃太多，熱量爆表不是一樣會胖？」

理論上是的。但實際上，你根本吃不下那麼多。

因為好澱粉纖維多、體積大、需要咀嚼很久、飽足感來得快。你試試看一次吃3碗糙米飯，根本吃不下。但3碗白飯拌炒飯？很容易就吃完了。

如何分辨好澱粉跟壞澱粉：3個判斷標準

不用死背清單。學會這3個標準，你就能自己判斷。

標準一：看加工程度

原則：越接近原型，越好

  • ✓ 好：整顆地瓜、糙米、燕麥片
  • ✗ 壞：地瓜片（油炸）、白米、即食燕麥（加糖）

標準二：看纖維有沒有被去掉

原則：纖維在，就是好澱粉

  • ✓ 好：全麥麵包、糙米、五穀飯
  • ✗ 壞：白麵包、蛋糕、餅乾

怎麼知道有沒有纖維？看營養標籤的「膳食纖維」：

  • 每100g有3g以上的纖維 → 不錯
  • 有5g以上 → 很好

標準三：想像血糖會不會暴衝

原則：能讓血糖緩慢上升的，就是好澱粉

  • ✓ 好：地瓜、糙米飯、豆類
  • ✗ 壞：蛋糕、白吐司、珍珠奶茶

簡單測試：

  • 如果吃完1-2小時後你又餓了 → 壞澱粉
  • 如果吃完3-4小時還很飽 → 好澱粉

如何吃澱粉不會胖：5個黃金原則

知道好壞澱粉還不夠，還要知道怎麼吃。

原則一：選原型，不選加工

  • ✓ 地瓜 > 地瓜球
  • ✓ 糙米 > 白米 > 稀飯
  • ✓ 蒸玉米 > 玉米濃湯
  • ✓ 燕麥 > 麥片（加糖的那種）

原則二：不要單吃澱粉

這個超重要！

白飯本身不是問題，問題是「只吃白飯」或「配菜太少」。

  • ✓ 白飯 + 大量蔬菜 + 蛋白質 = OK
  • ✗ 單吃一碗白飯 = 血糖暴衝
  • ✗ 一碗飯配一點菜 = 還是不夠

為什麼？因為蔬菜裡的纖維可以幫白飯「補上守門員」，蛋白質和油脂可以讓消化變慢，血糖就不會暴衝了。

原則三：把握黃金比例

一餐的理想比例：

  • 🥗 蔬菜：50%（至少大半碗，多多益善）
  • 🍗 蛋白質：30%（一個手掌大）
  • 🍚 澱粉：20%（一個拳頭大小）

重點：蔬菜一定要最多！不是「一點菜配很多飯」，是「很多菜配一點飯」

原則四：吃澱粉的時機

最好的時機：

  • ✓ 早餐：一定要吃足量澱粉（提供一整天的能量）
  • ✓ 午餐：正常吃，不要怕
  • ✓ 晚餐：可以少一點，但不是不吃
  • ✓ 運動後：可以多吃一點（這時候身體會拿去用，不會囤積）

不好的時機：

  • ✗ 宵夜吃澱粉（尤其是精緻澱粉）
  • ✗ 睡前2小時內吃大量澱粉

原則五：慢慢吃、多咀嚼

細嚼慢嚥可以：

  • 增加飽足感（大腦有時間接收「飽了」的訊號）
  • 減緩血糖上升速度
  • 幫助消化
  • 讓你更容易察覺「其實我已經飽了」

建議：

  • 每口至少咀嚼20下
  • 一餐至少吃20分鐘

4個調整方式，讓你聰明吃澱粉也能瘦

不用一次全改，太痛苦了。從這4個方向開始調整：

調整一：選對澱粉種類

把精緻澱粉換成原型澱粉：

  • 白吐司 → 地瓜、燕麥
  • 白麵 → 蕎麥麵、全麥麵
  • 可以考慮把白飯換成糙米飯（或一半一半）

不用一次全換，一餐一餐慢慢調整。

調整二：增加蔬菜比例

這個超級重要！

蔬菜的纖維可以幫澱粉「補上守門員」，讓血糖不會暴衝。

目標：每餐蔬菜至少要有半碗，最好大半碗。

如果外食便當菜不夠，自己買一盒燙青菜。

調整三：複合著吃，不要單吃

永遠記得：澱粉不要單獨吃。一定要搭配：

  • ✓ 蔬菜（纖維）
  • ✓ 蛋白質（增加飽足感）
  • ✓ 好油脂（讓消化變慢）

例如：

  • ✗ 單吃一碗白飯
  • ✓ 白飯 + 大量蔬菜 + 雞胸肉

調整四：調整吃澱粉的時機

不是不能吃，是要選對時間：

  • ✓ 早餐、午餐：正常吃，甚至可以多一點
  • ✓ 晚餐：可以少一點，但不是完全不吃
  • ✗ 宵夜：盡量避免澱粉（尤其精緻澱粉）

從這4個方向開始調整

不用急，慢慢來。你會發現：

  • ✓ 便秘改善了
